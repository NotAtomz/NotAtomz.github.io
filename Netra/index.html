Netra OS is a lightweight, browser-based interface for an ESP32 device. It is designed to feel like a modern mobile operating system while running locally over Wi‑Fi with no cloud required.

The project combines a polished app-style UI, device controls, live logs, settings, and hardware features into one compact experience that can be opened from a phone or tablet.

App-based interface

Netra OS uses a phone-like layout with app windows, navigation controls, and smooth transitions to make the experience feel familiar on mobile devices.

Runs locally

The system is designed to work over the ESP32-S3 own Wi‑Fi network, keeping everything self-contained and independent of internet connectivity.

Hardware features

The platform can expose device tools, settings, logs, storage, and other embedded functions through a clean interface that feels like a mini operating system.
